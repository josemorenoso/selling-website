## No Aplica

Este proyecto es un sitio web estático (HTML/CSS/JS) desplegado en Netlify.
No tiene base de datos propia.

El producto RestaurantQR que se muestra como showcase tiene su propia base de datos
(Supabase PostgreSQL) documentada en el repositorio separado de ese proyecto.

El dashboard demo en `restaurantqr.html` usa datos mock en JSON inline,
no se conecta a ninguna base de datos real.

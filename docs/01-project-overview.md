# Luis Moreno — Website & Product Showcase

## Visión
Sitio web multi-página de alta gama que posiciona a Luis Moreno como autoridad en automatización para negocios. Combina la oferta de servicios con un producto estrella (RestaurantQR — sistema de fidelización para restaurantes) como prueba de capacidad técnica. El objetivo principal es generar confianza, credibilidad y conversión a llamadas de venta vía Calendly.

## Objetivos
- Proyectar autoridad brutal: que el sitio parezca de $10,000 USD
- Integrar el producto RestaurantQR como showcase de capacidad técnica (con demo interactivo)
- Generar llamadas de venta de forma consistente (Calendly como CTA principal)
- Comunicar los servicios de forma ultra-simple, sin texto complejo
- Crear credibilidad y confianza inmediata en el visitante

## Producto Estrella — RestaurantQR
Plataforma full-stack de fidelización, CRM y automatización de marketing para restaurantes.
- **Registro QR** en mesa con flujo automatizado
- **Captación de domicilios** vía WhatsApp
- **Campañas automatizadas** (cumpleaños, reactivación) vía Twilio
- **Dashboard administrativo** completo con métricas, clientes, recompensas
- **Stack:** Next.js 16 + Supabase + TailwindCSS + shadcn/ui + Twilio
- **Estado:** MVP en producción (Vercel)

> Este producto se integra al sitio como prueba de autoridad técnica. El visitante puede ver un dashboard demo interactivo que demuestra la capacidad real de desarrollo.

## Principio Fundamental
> El sitio debe hacer que cualquier visitante piense: "Este tipo sabe lo que hace" — y la única acción natural sea agendar una llamada.
